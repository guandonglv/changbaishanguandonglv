# Changbai Mountain Guandonglv International Travel Website

Professional English travel website for Changbai Mountain inbound tours, built for overseas clients.

## Features
✅ Pure English interface for global tourists
✅ Responsive design (mobile/desktop)
✅ Blue & white professional travel theme
✅ Trust & qualification sections
✅ Tour product display
✅ Contact form & info
✅ SEO optimized

## Quick Start

1.  Place your images in `./assets/images/`:
    - hero-tianchi.jpg (main banner)
    - tour-1day.jpg (1-day tour image)
    - tour-2day.jpg (2-day tour image)
    - private-car.jpg (private car service image)

2.  Run local server:
    ```bash
    npm run dev
    ```

3.  Open browser and visit `http://localhost:55734`

## Tech Stack
- Bootstrap 5 (responsive framework)
- Font Awesome 6.4.0 (icons)
- Pure vanilla HTML/CSS/JS

## Customization

### Update Contact Info
Edit the contact section in `index.html`:
- WhatsApp/WeChat/Phone/Email

### Update Qualifications
Edit the footer and trust badge sections:
- Travel license number
- Business registration number

### Add Tour Products
Duplicate the tour card HTML block in the tours section and update content.
