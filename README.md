# 长白山保护开发区关东驴国际旅行社有限公司官网

基于GitHub Pages搭建的合规入境游网站，适配GYG政府审核要求。

## 快速部署步骤

1.  **注册GitHub账号**：https://github.com/
2.  **创建仓库**：点击右上角+ → New repository，仓库名填写 `changbaishan-guandonglv.github.io`
3.  **上传文件**：将本仓库所有文件上传到新建的GitHub仓库
4.  **开启GitHub Pages**：进入仓库Settings → Pages → Source选择Deploy from a branch → Branch选择main/(root) → Save
5.  **等待部署**：Actions标签页显示deployed即可访问网站

## 审核必备文件

请将以下文件上传到`assets/images`文件夹：
- `license-business.jpg`：营业执照扫描件
- `license-travel.jpg`：旅行社业务经营许可证扫描件
- `license-inbound.jpg`：入境旅游业务资质文件扫描件
- `license-guide.jpg`：专职导游资格证扫描件

## 自定义修改

1.  修改`_config.yml`中的网站名称、URL等信息
2.  编辑`_pages`下的页面文件修改内容
3.  替换`assets/images`中的示例图片为实际资质文件
4.  更新产品价格、联系方式等信息

## 合规说明

本网站包含以下合规内容：
- 经营资质公示（含入境游业务许可）
